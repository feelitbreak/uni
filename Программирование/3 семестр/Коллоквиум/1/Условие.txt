Дан текстовый файл input.txt, разделители между лексемами: пробел, запятая, точка с запятой, переход на новую строку. 

Может быть несколько строк. Файл может быть пустой.

Символ “точка” является разделителем целой и дробной части в вещественных числах.

Требуется найти сумму всех лексем-чисел, которые есть в данном входном файле.

Вывести сумму в выходной файл output.txt. 

Например, для текста: 

	Один 1, два 2, три 3 один 1      два 2; Много слов 123.456; 

	Минус один -1; 

Надо вывести: 

	131.456
